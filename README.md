# AI 小手机静态版

这是一个可直接上传到 GitHub Pages 的原创静态网站底包。

## 功能
- 手机式单页界面
- 本地聊天记录（localStorage）
- 角色名称 / 简介 / 头像文字设置
- 长期记忆文本
- 导入 / 导出 JSON 备份
- 页面打开期间的伪主动消息

## 上传到 GitHub Pages
1. 新建仓库
2. 上传整个目录中的文件
3. 在仓库 Settings -> Pages 中启用 GitHub Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main` 和 `/root`
6. 保存后等待生成访问链接

## 后续建议
- 把原网站中最重要的聊天记录和设定先导出，再导入这个站
- 再逐步替换文案、头像、背景、角色逻辑
- 真正后台主动消息要等后续 APK + 前台服务方案
